# App de Organização de Finanças Pessoais com IA

## Objetivo
Resolver o problema de controle financeiro manual e cansativo, oferecendo uma experiência interativa e inteligente para planejamento financeiro.

## Funcionalidades
- Chat com IA para perguntas sobre orçamento e metas.
- Dashboard inteligente com saldo, categorias e projeções.
- Planos automáticos de economia.
- Alertas proativos e insights personalizados.

## Como Usar
1. Clone este repositório.
2. Abra o arquivo PRD_App_Financas_IA.docx para entender os requisitos e prompts.
3. Utilize os prompts no Copilot para gerar interações.
4. Teste fluxos de tela no Lovable.
